# StyleLabor Launcher

A modern, cross-platform Minecraft launcher built with Tauri, React, and Rust. Lightweight, fast, and feature-rich.

---

## Features

- **Cross-Platform** — Windows and macOS support
- **Multi-Account** — Microsoft authentication with account switching
- **Modpack Support** — CurseForge, Modrinth, and Vanilla
- **Auto-Updates** — Built-in update system
- **Profile Management** — Multiple instances with custom settings
- **Discord Rich Presence** — Show what you're playing
- **Screenshot Gallery** — View and manage in-game captures
- **3D Skin Viewer** — Live preview of player skins

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React 19, TypeScript, TailwindCSS |
| Desktop | Tauri 2.0, Rust |
| State | Zustand |
| Animations | Framer Motion |
| UI | Lucide Icons, Sonner Toasts |

---

## Development

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [Rust](https://rustup.rs/)
- [Tauri CLI](https://tauri.app/)

### Setup

```bash
# Install dependencies
npm install

# Run in development
npm run tauri dev

# Build for production
npm run tauri build
```
